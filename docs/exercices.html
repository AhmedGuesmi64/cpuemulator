CP Emulator — Exercises
Short lab exercises and expected answers

Exercise 1 — Simple add

Goal: Add values at dmem[00] and dmem[01], store result in dmem[02].

LD [00], r0
LD [01], r1
ADD r2, r0, r1
ST r2, [02]

Set data memory 00=0x05 and 01=0x07, assemble and run. Expect dmem[02]=0x0C.

Exercise 2 — Carry and zero flags

Set dmem[00]=0xFF and dmem[01]=0x02 and run the same program as Exercise 1. Expect r2=0x01, Carry=1.

Exercise 3 — Simple loop (advanced)

Use data memory as a counter and loop until zero. Hint: emulate decrement by adding 0xFF to a register then store back. Requires careful sequence of LD/ADD/ST/JMP.
